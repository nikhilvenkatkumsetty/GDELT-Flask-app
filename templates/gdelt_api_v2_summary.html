<H3 STYLE="padding-left:15px;padding-bottom: 15px;">Step 3: Enter Search</H3>
<p CLASS="instructions-KEYWORDS hidden">Enter your search keywords below. All words/phrases should be in English, since you will be searching the English machine translations of all 65 live translated languages GDELT monitors (allowing you to search across languages). Phrases should be enclosed in quote marks. If you include multiple words/phrases all of them will be required to appear somewhere in a document for it to match. Only the exact keyword(s) entered are searched (searching for "russia" does NOT match "russian" or "russians"). You can perform limited boolean "OR" searches by enclosing a group of OR'd terms inside a set of parantheses - for example, to search for "syria" appearing near a set of russian-related terms you might search for "syria (russia OR russians OR russian OR kremlin OR putin)". You can also put a "-" in front of a given word or phrase to exclude articles that contain it. Remember that machine translation is far from perfect, so try to search for simple keywords and avoid phrases if possible to maximize results from non-English sources.</p>

<p CLASS="instructions-KEYWORDS hidden">In addition to searching textual news articles, three special advanced search terms can be used to visually and textually search online news imagery. Each day, GDELT processes up to one million global news images from the articles it monitors through Google's <a href="https://cloud.google.com/vision/">Cloud Vision API</a> deep learning algorithms, visually recognizing up to 10,000 distinct objects and activities, performing a reverse Google Images search to determine its popularity, identifying more than 2 million topics in most of the world's major languages from all captions used to describe the given image across the web, estimating how "happy" or "sad" the majority of the people in the image appear, transcribing background signage and text in 55 languages and extracting all embedded metadata, creating one of the most powerful visual search systems in the world for global news imagery.</p>

<p CLASS="instructions-KEYWORDS hidden">To add a visual search term to your search below, type <b>imagetag:"yourkeyword"</b> to search the 10,000 visual topics, type <b>imagetopic:"yourkeyword"</b> to search the 2 million textual caption topics or type <b>imagetext:"yourkeyword"</b> to search the combined raw caption text, embedded metadata, recognized logos and OCR'd background text. You must include the quote marks even when searching for a word rather than a phrase. Both imagetag:"" and imagetopic:"" will autosuggest terms from the predefined list of available tags/topics as you type the first letter after the opening quote mark. Try <b>imagetag:"protest"</b> or <b>imagetopic:"drone"</b> or combine with a textual search like <b>"climate change" imagetag:"protest"</b>.</p>

$(function() {

}

.autocomplete({

source: function( request, response ) {
//get the current term block and jump clear if it isn't one of our trigger terms...
var curpos = document.getElementById('k').selectionStart;
var type = 0;
var blockpos1 = request.term.toLowerCase().lastIndexOf('imagetag:"', curpos);
var blockpos2 = request.term.toLowerCase().lastIndexOf('imagetopic:"', curpos);
var blockpos = -1;
if (blockpos1 > blockpos2) { autocompleteglobal_blockstart = blockpos1; blockpos = blockpos1 + 10; type = 'imagetag'; }
if (blockpos2 > blockpos1) { autocompleteglobal_blockstart = blockpos2; blockpos = blockpos2 + 12; type = 'imagetopic'; }
if (blockpos == -1) { return false; } //neither of our trigger terms has been seen...
var thisblock = request.term.substring(blockpos, curpos);
if (thisblock.indexOf('"') > -1) { return false; } //whatever our last block was, we've finished it...

//proceed...
//jump clear until the user has typed at least 3 letters...
if (thisblock.length < 1) { return false; }
if (type == 'imagetag') {
var labels = lookup_imagetags.map(function(item){return {label: item, value: 'imagetag:"' + item + '"' };});
}
if (type == 'imagetopic') {
var labels = lookup_imagewebtags.map(function(item){return {label: item, value: 'imagetopic:"' + item + '"' };});
}
//limit ourselves to a max of 50 results...
results = $.ui.autocomplete.filter( labels, thisblock );
var resultscnt = results.length;
if (results.length > 50) { results = results.slice(0,50); results.push( {label: '... ' + (resultscnt-10) + ' more results ...', value: 'RESULTSCUTOFF' } ); }
response( results );
},
focus: function( event, ui ) {
if (ui.item.value == 'RESULTSCUTOFF') { return false; }
event.preventDefault(); //prevent key up/down from wiping out the contents of the search box...
},
select: function( event, ui ) {
if (ui.item.value == 'RESULTSCUTOFF') { return false; }
//splice our updated entry back into the box...
//get our updated location within the box (after the user has typed additional text)...
var curpos = document.getElementById('k').selectionStart;
var newval = this.value.substring(0, autocompleteglobal_blockstart) + ui.item.value + this.value.substring(curpos, this.value.length);
newval.replace(/""/g, '"'); //get rid of inadvertant double quotes... (happens when user tries changing the value of an existing tag)...
this.value = newval;

return false;
}
});
